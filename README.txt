3DMeteo

1) Descrizione

3DMeteo è un’applicazione web che mostra informazioni geografiche e meteo in modo interattivo. Permette di visualizzare i comuni italiani su una mappa, filtrare per regione o provincia e consultare i dati meteorologici aggiornati in tempo reale tramite il servizio Open Meteo.

2) Funzionalità principali

- Visualizzazione della mappa con Leaflet e OpenStreetMap

- Selezione dei comuni italiani e filtro per regione/provincia

- Visualizzazione dei dati meteo in popup e nelle card dell’interfaccia

- Marker interattivi sulla mappa per indicare le località

- Gestione dinamica della visuale della mappa con flyTo, setView e fitBounds

-Gestione di un comune preferito e un comune recente trameti localStorage

3) Tecnologie utilizzate

- HTML, CSS, JavaScript

- Leaflet per la mappa interattiva

- OpenStreetMap come base cartografica

- Fetch API per recuperare dati esterni in formato JSON

- File JSON dei comuni italiani

4) Come usare l’app

- Aprire index.html in un browser moderno (E' preferibile non usare Firefox, poiché essa non permette l'uso di localStorage con file locali)

- Selezionare una regione o una provincia dal menu

- Cliccare sulla mappa o sui marker per visualizzare i dati meteo

5) Note

- I dati meteo sono aggiornati in tempo reale tramite il servizio Open Meteo

- La mappa si adatta automaticamente alla località o ai marker selezionati
